# Conclusión

Este sprint ha sido el más completo del curso en volumen de trabajo y dificultad.
Hemos combinado conocimientos de DIW, DWEC y el Proyecto Intermodular de forma
simultánea, lo que nos ha permitido ver cómo cada módulo aporta algo concreto al
proyecto final.

En **DIW** hemos dado un paso importante hacia el desarrollo web profesional, aplicando
criterios de SEO, accesibilidad y usabilidad que van más allá de la maquetación visual.

En **DWEC** hemos mejorado en el uso de React y asumido responsabilidades nuevas como
el testing y la documentación de usuario.

A nivel de equipo, la huelga del profesorado y algunos problemas técnicos con la API
nos han puesto a prueba, pero hemos sabido organizarnos y sacar el trabajo adelante.

Como equipo, cerramos este sprint con un proyecto más sólido, mejor documentado y más
cercano a lo que sería un producto real.
