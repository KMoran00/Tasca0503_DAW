## Introducción

Durante este sprint hemos avanzado en los tres módulos principales del curso: Diseño de Interfaces Web (DIW), Desarrollo en Entorno Cliente (DWEC) y Proyecto Intermodular (PI).

En DIW hemos profundizado en el uso de metaetiquetas, SEO técnico, Bootstrap local, tipografías y técnicas de usabilidad, culminando con una actividad práctica de maquetación orientada al examen. En DWEC hemos consolidado el uso de React, llevado a cabo el testing del proyecto y elaborado un manual de usuario. Todo ello se ha integrado progresivamente en el Proyecto Intermodular, mejorando tanto la tienda fake como el comparador y el backoffice.
